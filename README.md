# Tales App — Public Assets

Generated illustrations, audio narrations, and story JSON files for the **Tales App** children's storybook project. Served via **jsDelivr CDN** to the Flutter app.

> 🔒 The Flutter app source code lives in a separate private repository. This repo contains only the consumable public-domain content assets.

## Content

- **`stories/`** — 47 JSON story files (Arabic + English narration text + image prompts), plus `manifest.json` for catalog browsing
- **`images/`** — 556 WebP illustrations (1024×1024), generated by Cloudflare Workers AI (Flux Schnell) and Pollinations.ai (Sana)
- **`audio/`** — 1242 MP3 narrations (AR + EN per scene), generated by Microsoft Edge TTS via [edge-tts](https://github.com/rany2/edge-tts)

## CDN URL pattern

```
https://cdn.jsdelivr.net/gh/aboassar13-ui/tales-app-assets@main/images/<story_id>/<scene_id>.webp
https://cdn.jsdelivr.net/gh/aboassar13-ui/tales-app-assets@main/audio/<story_id>/<scene_id>_<lang>.mp3
https://cdn.jsdelivr.net/gh/aboassar13-ui/tales-app-assets@main/stories/<story_id>.json
```

Fallback:
```
https://raw.githubusercontent.com/aboassar13-ui/tales-app-assets/main/...
```
